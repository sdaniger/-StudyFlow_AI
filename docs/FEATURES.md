# Features

## Tasks
- Create, edit, delete tasks
- Mark as complete (generates review schedule)
- Subject classification
- Priority levels (low, medium, high, urgent)
- Due date tracking
- AI-generated task decomposition

## Pomodoro Timer
- 25min focus / 5min short break / 15min long break
- Custom timer durations
- Start, pause, reset controls
- Session counter
- Circular timer visualization

## Habit Tracker
- Create daily habits with subject tags
- Check-in completion
- Streak calculation
- Frequency settings (daily, weekly, weekday, weekend)

## Review System
- Forgetting-curve based scheduling (1, 3, 7, 14, 30 days)
- Auto-generated on task completion
- Confidence rating (1-5)
- Today's review dashboard

## AI Planner (Mock)
- Goal input form
- Generated study plan with daily/weekly tasks
- Preview before accepting
- One-tap add to task list

## Calendar Sync (Mock)
- Suggest events from tasks
- Preview event drafts
- Register/edit/cancel UI

## Drive Sync (Mock)
- Backup app data
- Restore from backup
- Sync status display
